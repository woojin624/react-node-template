# 리액트와 노드Js Express 연동 템플릿입니다. ✨

프론트엔드 개발을 하면서 리액트를 써서 만드는 경우가 많았는데
이 경우 create-react-app을 써서 만들면 바벨과 웹팩이 세팅된 상태로 프로젝트를 만들 수 있었습니다.
